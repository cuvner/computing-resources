## Key Knowledge – What is a network?

### Types of Networks – Notes

Standalone computers are not connected to any network.

**Benefits of networking computers:**

- Share files and peripherals (e.g., printers, internet connection).
- Access personal files from any computer on the network.
- Servers can manage security, software updates, and backups.
- Easier communication (e.g., email, social media).

**Local Area Network (LAN):**

- Covers a small area (e.g., one building).
- Hardware is owned by the organisation.
- Connections use UTP cables, fibre-optic cables, or Wi-Fi.

**Wide Area Network (WAN):**

- Covers a large area (e.g., between cities).
- Connects multiple LANs together.
- Infrastructure (like cables) is hired from telecom companies.
- Uses telephone lines, fibre-optic cables, or satellite links.

**Network devices:**

- LANs use switches to connect computers.
- WANs use routers to connect networks.
- Home routers often combine router, switch, and wireless access point in one device.
- Large networks usually have separate devices for each job.

**Disadvantages of networking:**

- Higher security risks as data is shared.
- Malware and viruses can spread easily.
- If a central server fails, computers may lose access to services.
- Networks can become slow if too much data travels at once.

## Key Knowledge – Network Hardware

The Internet is fundamentally a physical infrastructure consisting mainly of wires (such as fibre optics or copper cables) and, less commonly, satellite or cellular connections. This physical medium enables communication between computers through a system of addressing and routing.

### Servers

- A server is a specialised computer that is connected directly to the Internet.
- Servers store web pages, files, and other digital resources on their hard drives and are accessible to other devices on the Internet.
- Each server is identified by a unique Internet Protocol (IP) address, functioning analogously to a postal address, to facilitate accurate location and communication (e.g., 72.14.205.100).
- For human convenience, domain names (e.g., google.com) are assigned to these numeric IP addresses via the Domain Name System (DNS).

### Clients

- Clients refer to the computers and devices that people use in everyday life (e.g., laptops, smartphones).
- Unlike servers, clients are not directly connected to the Internet but instead access it through an Internet Service Provider (ISP).
- Clients initiate communication with servers to request information, such as web pages or emails.

### Routers

- Routers are network devices located at intersections within the Internet's infrastructure.
- Their primary function is to direct packets of data toward their intended destination by reading the IP address information attached to each packet.
- When information (e.g., a web page or an email) is transmitted across the Internet, it is broken into packets. Each packet is wrapped in layers of addressing information, with each router adding or removing a layer to help the packet move closer to its final destination.
- A typical request, such as accessing a website, may involve 10 to 15 routers guiding the packets back and forth between the client and server.
