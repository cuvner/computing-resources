# Lesson 2 – How to Buy The Right Computer

## In this lesson

In this lesson, you'll learn about the key components of a computer and how to choose a computer that will be able to do everything it needs to do.

## Learning Outcomes

- I can explain what CPUs, Memory, and Storage are and the roles they play in making a computer work.
- I can choose a computer that is suitable for a given set of requirements and job role.

## Key Knowledge

### 🧠 What Do Computers Need to Work Well?

To understand how a computer works, you need to know about three main parts:

### 1. CPU – Central Processing Unit

**What it does:**
The CPU is like the brain of the computer. It carries out instructions and makes the computer run. Every time you open an app or play a game, the CPU is doing the thinking.

**Key Performance Factors:**

- **Clock Speed (GHz):** This tells you how fast the CPU can process instructions. A higher number means it can do more in less time.
- **Number of Cores:** Each core is like a mini-CPU. More cores means the computer can do more things at the same time (good for multitasking or running big programs).

### 2. Memory – RAM (Random Access Memory)

**What it does:**
RAM is like the computer's short-term memory. It stores the things you're working on right now. The more RAM you have, the more programs or browser tabs you can have open at once without the computer slowing down.

**Key Performance Factors:**

- **Size (GB):** This tells you how much RAM the computer has. More RAM = better multitasking, can play more demanding games, and smoother performance.

### 3. Storage – Hard Drive or SSD

**What it does:**
Storage is like the long-term memory of the computer. It's where all your files, apps, photos, and games are saved—even when the computer is turned off.

**Types of Storage:**

- **HDD (Hard Disk Drive):** Older and usually cheaper, but slower.
- **SSD (Solid State Drive):** Newer, faster, and more durable but more expensive.

**Key Performance Factors:**

- **Size (GB or TB):** How much space you have for files and software.
- **Read/Write Speed:** How quickly the computer can open and save files. SSDs are much faster at this than HDDs.

### 4. Graphics Card – GPU (Graphics Processing Unit)

**What it does:**
The GPU handles all the visuals on your screen. It's especially important for anything that involves high-quality images, video, animation, or 3D graphics—like editing video, playing games, or using design software. While basic tasks can use the CPU for graphics, more advanced work needs a dedicated graphics card.

**Key Performance Factors:**

- **Video Memory (VRAM):** Like RAM, but for the graphics card. More VRAM lets the GPU handle more detailed images and higher resolutions without slowing down.
- **Clock Speed:** Like the CPU, this affects how fast the GPU can process data.
