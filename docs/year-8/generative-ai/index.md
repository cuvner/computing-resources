# Introduction to Generative AI

*Year 8 · 4 lessons*

We'll be looking at Generative AI. What can it be used for? What do we need to look out for when we're using it? And how can we use it to support our learning?

## Lessons

1. **Safely Using Generative AI** — what generative AI is; the rules for safe use; hallucinations
2. **Using Generative AI to Learn** — question mapping; learning reliably with LLMs
3. **Creative AI** — generating film synopses and posters with LLMs and image generators
4. **Using Generative AI Responsibly** — misinformation vs disinformation; the REVIEW method
