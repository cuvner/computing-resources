# Lesson 1a — Intro to Generative AI

**In this course:** We'll be looking at Generative AI. What can it be used for? What do we need to look out for when we're using it? And how can we use it to support our learning?

**In this lesson:** You'll get up and running with a "Large Language Model", which is a generative AI program that outputs text in response to a "prompt".

## Learning Outcomes

- I can use a large language model to create a text output in response to a prompt.
- I can define the key words "generative AI", "prompt", "large language model", and "chatGPT".
