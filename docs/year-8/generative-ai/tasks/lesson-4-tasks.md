## Task 4.1: Generating Mis- and Dis- Information

Generating Misinformation. In this activity, we'll see how easy it is to create convincing misinformation. We'll use ChatGPT to create an entirely false news report. The news report can be about whatever you like, however it must also be believable enough to be convincing and appropriate for school.


To Create Your Fake News Report:

1. Open this Google Drawing [LINK TO CAPTURE - see manifest]

2. Make your own copy of it (File > Make a copy)

3. Open a LLM and use the prompt below to help you create it: "Create a news report about [your idea here]"

Remember: you can have a dialogue with the LLM, if you don't like the first thing it comes up with then ask it to change it!

4. Use Canva to create a GenAI 'photo' to accompany the news story.

5. In your Google Drawing news report: File > Download > PNG Image

6. Click Insert Image below and upload your news report image.
