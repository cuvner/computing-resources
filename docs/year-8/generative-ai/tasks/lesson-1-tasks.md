## task 14 seeing hallucinations in practice

Generative AI "Hallucinates" A "hallucination" is when people see things that aren't real. When Generative AI makes a mistake, we call it a "Hallucination".Why a "hallucination"?Because when Generative AI programs make mistakes, they do so confidently. They don't ask questions if they're unsure, they just give you a response even if it's wrong.The confidence with which Generative AI programs give you the wrong answer can make it really convincing.In order to avoid falling for a hallucinationthis, you need to make sure you are cross referencing anything a generative AI programtells you with a reliable source.Hallucination Example:You can see a visual example of hallucination below. I asked stable diffusion (a Generative AI image generator) for a "flying pink elephant". Can you spot the errors?
