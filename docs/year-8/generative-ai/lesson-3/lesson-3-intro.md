# Lesson 3 — Using Generative AI to Be Creative

**In this lesson:** We'll be looking at how we can use large language models (like ChatGPT) to be creative.

## Learning Outcomes

- I can use a Large Language Model to generate a title and synopsis of a film.
- I can use an image generator to create relevant media, and include it in a film poster I have made.
